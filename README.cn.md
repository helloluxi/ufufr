# Lu Timer（三盲训练器）

[![en](https://img.shields.io/badge/lang-English-blue.svg)](README.md)
[![简体中文](https://img.shields.io/badge/lang-简体中文-red.svg)](README.cn.md)

一款专为三盲设计的网页训练工具，采用自适应练习方法，针对薄弱环节进行重点训练，提升学习效率。

本项目更像是一个代码库而非成品。我鼓励你阅读代码，理解其背后的原理，并根据自己的训练需求构建适合自己的版本。

## 核心理念

**太长不看：** 模拟比赛 ≠ 高效训练，薄弱case需要更多练习机会。

在比赛中，打乱是完全随机的——每个case出现的概率相等（当然根据 cycle break 策略会略有差异）。但这种随机性对于训练来说效率并不高。如果你已经熟练掌握了 818 种case中的 800 种（秒记、零失误），但还有 18 种特定case始终不够熟练（记忆慢或偶尔出错），那么随机打乱就会让你把大量时间浪费在反复练习已经掌握的case上。当然，偶尔用完全随机的打乱来保持手感仍然是有必要的。

**核心思想：** 自适应频率训练效率更高。不熟练的case应该比已掌握的case更高频率地出现。本工具实现了这个理念——你可以标记不熟练的case，它们会在后续训练中自动提高出现频率。这个理念借鉴了打字练习网站 [keybr.com](https://www.keybr.com/)，以及运动员、音乐家、医生普遍使用的刻意练习原则。

## 工作原理

训练算法采用加权采样系统：

- **练习过的case**：`times^(-2/(1 + mark))`
- **新case**：你设定的学习率

权重函数更像是一个经验公式而非严格测试结果，你可以根据自己的学习习惯随意调整。

## 快速开始

### 安装

1. **Fork 本仓库** - 强烈推荐，方便做版本控制和数据管理。
2. 将你 fork 的仓库克隆到本地，可以选择本地使用 / 部署到 Github Pages / 部署到自己的服务器（搭配你喜欢的数据同步方案！）

### 个性化配置（可选）

[alg.txt](alg.txt) 和 mem.txt 均使用 `KEY=VALUE` 格式——每行一条，空行和不含 `=` 的行会被忽略：

```txt
CF=coffee
DN=donut
```

- **mem.txt** — 每个字母对的记忆词或图像提示
- **alg.txt** — 每种case的还原公式（原样存储和显示）

也可以通过命令行 `a KEY=VAL`、`m KEY=VAL` 逐条修改，无需重新上传整份文件。

其他设置如编码方案、缓冲块位置和颜色配置可以在设置面板中配置。

## 使用指南

### 计时器使用

- **启动计时器**：长按魔方显示区域或长按空格键直到变绿
- **停止计时器**：计时过程中轻触魔方显示区域或按任意键停止

### Case 统计格式

在命令行中使用 `.KEY` 命令查询 case 时，显示的统计数据格式为：
- **格式**：`标记次数/练习次数 [权重]`
  - `标记次数`：你标记此 case 为困难的次数
  - `练习次数`：你练习此 case 的总次数
  - `权重`：当前采样权重（越高 = 出现概率越大）

示例：`3/45 [0.012]` 表示标记 3 次，练习 45 次，权重 0.012

设置面板中也显示相同的统计数据：
- `xxx/818`：你见过的唯一 case 数量
- `Freq`：每个 case 的平均练习次数
- `Marks`：已标记 case 数量 / 总标记次数

### 标记系统

标记某个case时（点击对应按钮）：
- 按钮外观会改变（高亮显示，鼠标悬停时显示标记次数）
- 该case在后续训练中出现频率提高
- 权重增加：标记次数越多 = 出现概率越高
- 统计数据自动保存至浏览器 localStorage

一些case暂不支持标记：
- 浮动三循环
- 奇偶（有小循环和翻色时）
- 随机模式

### Case 按钮角标

练习界面中的按钮使用角标颜色表示 case 状态：

- **橙色角标**：出现在当前打乱中的 case
- **绿色角标**：你标记为需要练习的 case（鼠标悬停可查看标记次数）

这些角标帮助你快速识别哪些 case 需要练习，哪些在当前打乱中。

### 命令行界面

工具内置了用于高级操作的命令行。打开方式：
- 双击魔方显示区域
- 或直接开始输入（会自动打开）

`h` 会打开教程中的命令帮助页（当前为第 4 页）。

| 命令 | 说明 | 示例 |
|---------|-------------|---------|
| `.KEY` | 查询特定case的信息 | `.CG` 显示 CG 的记忆编码、公式和统计数据 |
| `+KEY1 KEY2 ...` | 手动标记case | `+CG c+g+` 标记 CG 和 c+g+ |
| `-KEY1 KEY2 ...` | 移除标记并重置 | `-CG` 将 CG 重置为未标记状态 |
| `m KEY=VAL` | 保存/更新单条记忆词 | `m CG=coffee` |
| `a KEY=VAL` | 保存/更新单条公式 | `a CG=[R U R',D]` |
| `a KEY=VAL&` | 保存公式并为对称字母对自动生成交换子逆向 | `a CG=[R U R',D]&` |
| `c NUM` | 复制最近 `NUM` 条成绩（时间、打乱、时间戳），默认 `1` 条 | `c 5` |
| `d MODE` | 切换训练模式 | `d e` 切换到纯棱训练模式 |
| `t CAT` | 显示标记次数最多的前 7 个case（可按类别筛选） | `t par` |
| `e` / `i` | 打开文件面板（导出/导入） | `e` |
| `h` | 打开教程中的终端帮助页 | `h` |
| `u` | 重新加载 service worker | `u` |

按 `Enter` 执行命令，按 `↑` 键可调用上一条命令。

训练模式：
- `r`: 随机模式
- `e`: 纯棱训练
- `c`: 纯角训练
- `ec`: 棱角分控（棱块+角块均自适应）
- `rc`: 棱随角控（棱块随机+角块自适应）

### 快捷键

- `Tab`：生成下一个打乱
- `` ` ``：打开/关闭设置面板
- `1`-`9`：标记/取消标记对应按钮（从左到右）
- `↑`：在命令行中调用上一条命令
- `Enter`：在命令行中执行命令
- `Esc`：关闭命令行或设置面板

### 数据存储

所有数据存储在浏览器 localStorage 中：
- **已看教程**：是否已查看教程（`bld.seenTutorial`）
- **统计数据**：每种case的练习次数和标记次数（`bld.stats`）
- **个人设置**：用户偏好配置（`bld.settings`）
- **公式与记忆**：上传或命令编辑后的内容（`bld.alg`、`bld.mem`）
- **成绩历史**：最近 100 条成绩（`bld.history`）

本工具不会上传任何数据，你的数据完全留在本地设备上。

### 离线支持与 Service Worker

本应用使用 Service Worker 实现离线功能。首次加载后，即使断网也能继续使用。

- 有网络时刷新页面会自动更新缓存文件
- 使用命令行中的 `u` 命令可手动触发缓存更新
- Service Worker 会自动清理旧缓存以节省空间

### 渐进式 Web 应用 (PWA)

本应用可以作为渐进式 Web 应用安装：

- **电脑端**：点击浏览器地址栏中的安装图标
- **移动端**：使用浏览器菜单中的"添加到主屏幕"
- 安装后，即使离线也能使用，体验类似原生应用
- 更新由 Service Worker 自动管理
