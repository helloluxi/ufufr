🖥️ Terminal Commands

Double-click the cube (or just start typing) to open.

.KEY      look up memo, alg, and stats for a case
+KEY      mark a case as difficult
-KEY      unmark / reset a case
m KEY=VAL save a memo word for a letter pair
a KEY=VAL save an algorithm for a case
e / i     export / import all data (opens Files panel)
h         open this help panel
d MODE    change mode (r / e / c / ec / rc)
t CAT     show top 7 most marked cases (filter by category)
c NUM     copy latest NUM solves to clipboard (default: 1)
u         force refresh cached files

You can always return here with the h command or from Settings.

🚀 Quick Start Guide

  • Open Settings: Double-click or tap the scramble text / press ` key
  • Start Timer: Long-press the cube / press space
  • Next Scramble: Press Tab / click "Next →"
  • Mark Cases: Click buttons to mark hard cases for more frequent practice
  • Important: Scramble in your solving orientation, not WCA default (white-green)
